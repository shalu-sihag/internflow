# InternFlow

A React-based Job Application Tracker that helps students manage internship applications, interview preparation, deadlines, and application progress.

## Features

- Add and manage applications
- Status tracking
- Search and filters
- Deadline reminders
- Interview preparation checklist
- Progress tracking
- Dark/Light mode
- LocalStorage persistence
- Responsive design

## Tech Stack

- React
- React Router
- Context API
- JavaScript
- CSS
- LocalStorage
